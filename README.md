## 🚀 部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 推送代码：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin <your-repo-url>
   git push -u origin main
   ```
3. 在 GitHub 仓库设置中启用 GitHub Pages：
   - Settings → Pages
   - Source: Deploy from branch
   - Branch: main / root
